# Hospital Emergency Dashboard

## Project Overview
This project presents an interactive **Hospital Emergency Dashboard** built using **Power BI**.  
It analyzes patient admissions, waiting time, demographics, and departmental referrals to help hospitals improve operational efficiency and decision-making.

---

## Tools & Technologies
- Power BI
- Power Query
- DAX
- Microsoft Excel / CSV

---

## Data Cleaning & Transformation
- Standardized gender values (M → Male, F → Female, NC → Not Confirmed)
- Created patient name using first-name initial and last name
- Split admission date into date and time columns
- Built a Date Table for time-based analysis

---
