# Hospital Emergency Dashboard

## Project Overview
This project presents an interactive **Hospital Emergency Dashboard** built using **Power BI**.  
It analyzes patient admissions, waiting time, demographics, and departmental referrals to help hospitals improve operational efficiency and decision-making.
